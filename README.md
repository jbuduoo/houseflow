# 🏠 住通地圖全自動開發系統 (HouseFlow Automation System)

這是一套針對「住通地圖」開發的自動化資料採集與補全系統。它能將地圖上的物件資訊轉化為詳細的開發清冊，包含自動抓取戶籍地址、OCR 辨識影像地址，以及透過 ArcGIS API 補全精確經緯度座標。

---

## 📂 目錄結構說明

### 🚀 啟動區 (根目錄)
*   **`0_chrome.bat`**：啟動具備遠端偵錯功能 (Port 9222) 的 Chrome 瀏覽器。
*   **`1_master_run.py`**：**【核心總控】** 自動化流水線。執行一次即可完成「採集 -> 地址補完 -> 戶籍補完 -> 座標定位」全流程。
*   **`2_web_house.py`**：**【地圖視覺化】** 使用 Streamlit 建立的房仲攻堅地圖，可直觀查看物件與屋主戶籍地的關聯。
*   **`houseflow_gheet_key.json.json`**：Google 試算表 API 金鑰（請妥善保管）。

### 🛠️ 核心組件 (core/)
*   `1_house.py`：負責從住通地圖採集初步物件清單。
*   `2_address.py`：深入物件頁面抓取比對地址與謄本連結。
*   `3_registry.py`：【最強組件】負責解析謄本、執行 OCR 辨識、地址清理及自動座標定位。
*   `4_coords.py`：獨立的 ArcGIS 座標轉換工具。

---

## 🛠️ 標準作業程序 (SOP)

請按照以下順序執行以確保系統正常運作：

### 第一步：環境準備
1. 雙擊執行 **`0_chrome.bat`**。
2. 在開啟的 Chrome 視窗中，**手動登入** 住通系統。

### 第二步：執行自動化補完
1. 開啟終端機 (PowerShell)，輸入：
   ```bash
   python 1_master_run.py
   ```
2. **階段 1 (採集)**：程式會停在搜尋頁面。請在地圖上搜尋您要開發的區域後，回到終端機按 **Enter**。
3. **自動接力**：程式會自動完成後續的地址補完、OCR 辨識與座標查尋。

### 第三步：查看地圖成果
1. 補完完成後，輸入：
   ```bash
   streamlit run 2_web_house.py
   ```
2. 您的瀏覽器會自動跳出「房仲攻堅地圖」，即可開始分析點位。

## ✨ 核心技術特色
*   **智能地址清理**：自動移除「里/鄰/村」等干擾定位的資訊，並修正縣市升格名稱。
*   **影像辨識 (OCR)**：針對圖片形式的地址自動轉為文字，並標註 `(OCR)` 供人工後校。
*   **雙重座標補完**：同時計算「案件物件點」與「屋主戶籍點」，並在地圖上以紅綠色區分。
*   **斷點續傳**：所有程式皆會偵測試算表空值，只針對缺少的資料進行補完，節省時間。
