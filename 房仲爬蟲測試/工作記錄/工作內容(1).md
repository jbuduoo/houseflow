# 工作內容 1 - 執行成果

## 🛠️ 執行細節
- **分析網站結構**：確認信義房屋的經緯度資料儲存在頁面原始碼的 `<script>` 區塊中（`lat` 與 `lng` 欄位）。
- **編寫 Python 爬蟲** (`sinyi_scraper.py`)：
    - 自動讀取 `名單` 或 `名單.md` 中的網址。
    - 過濾出信義房屋的連結（並自動修正如 `ttps://` 的格式問題）。
    - 提取經緯度並輸出至 log 檔案。
- **產出 Log 檔案** (`scrape_log.txt`)：格式為 `網址 > 經緯度`。
- **驗證重複地址**：程式在執行完畢後會自動檢查是否有重複的經緯度，並在終端機提示。

## 📊 執行結果摘要
- **總共爬取**：5 筆信義房屋網址。
- **重複檢查**：發現以下兩筆網址使用相同的經緯度 (`25.00258107108, 121.52226700325`)：
    - https://www.sinyi.com.tw/buy/house/6294NF?breadcrumb=list
    - https://www.sinyi.com.tw/buy/house/1459QF?breadcrumb=list
